# Predictor26

Predictor26 is a full-stack TypeScript workspace.

## Stack

- Angular 21 frontend in `apps/web`
- Express 5 API in `apps/api`
- SQLite local database in `apps/api/data`
- npm workspaces from the repository root

## Commands

```bash
npm install
npm run dev
npm run build
```

Development URLs:

- Web: `http://localhost:4200`
- API health: `http://localhost:3000/api/health`
